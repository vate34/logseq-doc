- **Java 中有 8 个基本类型，分别对应的 8 个包装类**
	- byte -- Byte
	- boolean -- Boolean
	- short -- Short
	- char -- Character
	- int -- Integer
	- long -- Long
	- float -- Float
	- double -- Double
- **为什么需要包装类**:
	- 基本数据类型方便、简单、高效，但泛型不支持、集合元素不支持
	- 不符合面向对象思维
	- 包装类提供很多方法，方便使用，如 Integer 类 toHexString(int i)、parseInt(String s) 方法等等
